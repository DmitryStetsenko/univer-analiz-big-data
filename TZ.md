# Технічне завдання: Багатосторінковий React-застосунок "Аналіз та обробка великих даних"

## Загальний опис
Застосунок призначений для супроводу навчального курсу "Аналіз та обробка великих даних". Він має бути сучасним, інтерактивним, зручним для користувача та оптимізованим для розгортання на GitHub Pages.

## Технологічний стек
- **Фреймворк**: React (Vite)
- **Маршрутизація**: React Router (використовувати `HashRouter` для сумісності з GitHub Pages)
- **Менеджер стану**: Zustand
- **UI-компоненти**: Ant Design (antd)
- **Візуалізація графіків/схем**: Recharts або кастомні SVG-компоненти
- **Мова**: JavaScript / React JSX

## Структура застосунку та вимоги до сторінок

### 1. Сторінка "Лабораторна робота №1" (Обчислення мір близькості для числових даних)
- **Завдання 1: Розрахунок мір близькості**
  - Вивести вхідні координати для об'єкта 5 $(x_1=2, x_2=4)$ та об'єкта 7 $(x_1=5, x_2=10)$.
  - Розрахувати та вивести 5 мір близькості:
    1. Відстань Евкліда
    2. Квадрат відстані Евкліда
    3. Манхеттенська відстань
    4. Відстань Чебишева
    5. Пікова відстань (Power distance)
  - Відображати математичні формули для кожної міри.
- **Завдання 2: Побудова матриці відстаней**
  - Для чотирьох об'єктів (об'єкти 1, 2, 3, 8) побудувати симетричну квадратну матрицю відстаней з нулями на головній діагоналі.
  - Метрика: Відстань Евкліда.
  - Дані мають відображатися у вигляді інтерактивної таблиці.
- **Джерело даних**: Об'єкт `lab1` з файлу `datasets.json`.

### 2. Сторінка "Лабораторна робота №2" (Ієрархічна кластеризація: Метод найближчого сусіда)
- **Алгоритм**: Ієрархічна агломеративна кластеризація (одиночний зв'язок). При об'єднанні рядків/стовпців вибирається *мінімальне* значення відстані.
- **Функціонал**:
  - Покрокова візуалізація зменшення матриці відстаней на кожному кроці ітерації.
  - Можливість перемикання кроків (вперед/назад) для детального аналізу.
- **Візуалізація**: Побудова деревоподібної дендрограми, яка відображає об'єднання кластерів та відстані між ними.
- **Джерело даних**: Об'єкт `labs2_3` з файлу `datasets.json`.

### 3. Сторінка "Лабораторна робота №3" (Ієрархічна кластеризація: Метод найдальшого сусіда)
- **Алгоритм**: Ієрархічна агломеративна кластеризація (повний зв'язок). При об'єднанні рядків/стовпців вибирається *максимальне* значення відстані.
- **Функціонал**: Покрокова візуалізація та дендрограма (аналогічно до Лабораторної №2).
- **Джерело даних**: Об'єкт `labs2_3` з файлу `datasets.json`.

### 4. Сторінка "Теорія: Вижимка лекцій"
Інтерактивний конспект з інфографікою, що охоплює теми:
- **Шкали вимірювань**: номінальна, порядкова, інтервальна, відносна (з допустимими математичними операціями).
- **Міри близькості**: порівняння матриці подібності та матриці несхожості (відстаней).
- **Кластеризація**: відмінність між агломеративними (знизу вгору) та дивізимними (зверху вниз) алгоритмами ієрархічної кластеризації.

### 5. Сторінка "Екзаменаційні питання"
- Інтерфейс у вигляді акордеона (FAQ) для 18 питань.
- Для питань 13-18 (які стосуються кластеризації та матриць близькості) надати короткі детальні відповіді та супроводжувати їх візуальними схемами/таблицями.

## Особливості розгортання на GitHub Pages
1. Використання `HashRouter` замість `BrowserRouter` в React Router для уникнення помилок 404 при перезавантаженні сторінок.
2. Налаштування відносного шляху (`base: './'`) у файлі `vite.config.js`.
3. Додавання скриптів для деплою (`gh-pages`) у `package.json`.
